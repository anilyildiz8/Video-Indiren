# Video Indiren

A modern, fast, and user-friendly desktop video downloader localized in Turkish.


-   FastAPI (Python)
-   yt-dlp
-   HTML5, Vanilla CSS3 (Glassmorphism design), JavaScript (ES6+)
-   Uvicorn
-   PyInstaller
-   FFmpeg (Auto-downloaded if missing)
